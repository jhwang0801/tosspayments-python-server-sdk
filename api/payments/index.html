    결제 API

    결제 승인, 조회, 취소 등 결제와 관련된 모든 기능을 제공합니다.

    클라이언트 접근

    from tosspayments_server_sdk import Client

client = Client(secret_key="your_secret_key")

# payments 리소스에 접근
payments = client.payments

    결제 승인

    결제창에서 받은 정보로 실제 결제를 승인합니다.

    기본 사용법

    payment = client.payments.confirm(
    payment_key="mc_1234567890123456789012345678901234567890",
    order_id="order-20240101-001",
    amount=15000
)

    매개변수

    이름 타입 필수 설명
    payment_key str ✅ 결제키 (결제창에서 받은 값)
    order_id str ✅ 주문 ID (가맹점에서 생성한 고유값)
    amount int ✅ 결제 금액

    응답

    Payment 객체가 반환됩니다. 주요 필드:

    print(payment.status)           # PaymentStatus.DONE
print(payment.method)           # "카드" 
print(payment.total_amount)     # 15000
print(payment.order_name)       # "상품명"
print(payment.approved_at)      # datetime 객체

    에러 처리

    from tosspayments_server_sdk.exceptions import ValidationError, APIError

try:
    payment = client.payments.confirm(
        payment_key=payment_key,
        order_id=order_id, 
        amount=amount
    )
except ValidationError as e:
    print(f"입력값 오류: {e}")
except APIError as e:
    print(f"API 오류 [{e.error_code}]: {e.message}")

    결제 조회

    paymentKey로 조회

    payment = client.payments.retrieve("mc_1234567890123456789012345678901234567890")

    orderId로 조회

    payment = client.payments.retrieve_by_order_id("order-20240101-001")

    결제 상태 확인

    # 상태 확인 메서드들
if payment.is_paid():
    print("결제 완료")

if payment.is_canceled():
    print("결제 취소됨")

if payment.can_be_canceled():
    print(f"취소 가능 금액: {payment.get_cancelable_amount()}")

    결제 취소

    전체 취소

    canceled_payment = client.payments.cancel(
    payment_key="mc_1234567890123456789012345678901234567890",
    cancel_reason="단순 변심"
)

    부분 취소

    canceled_payment = client.payments.cancel(
    payment_key="mc_1234567890123456789012345678901234567890",
    cancel_reason="부분 환불", 
    cancel_amount=5000  # 5,000원만 취소
)

    가상계좌 환불 (환불 계좌 지정)

    canceled_payment = client.payments.cancel(
    payment_key="mc_1234567890123456789012345678901234567890",
    cancel_reason="환불 요청",
    refund_receive_account={
        "bank": "88",  # 신한은행
        "accountNumber": "12345678901", 
        "holderName": "홍길동"
    }
)

    취소 매개변수

    이름 타입 필수 설명
    payment_key str ✅ 결제키
    cancel_reason str ✅ 취소 사유
    cancel_amount int ❌ 취소 금액 (미지정시 전체 취소)
    refund_receive_account dict ❌ 환불 계좌 정보
    tax_free_amount int ❌ 면세 금액

PaymentStatus.EXPIRED              # 결제 만료

    def process_payment(payment_key: str, order_id: str, amount: int):
    try:
        # 1. 결제 승인
        payment = client.payments.confirm(
            payment_key=payment_key,
            order_id=order_id,
            amount=amount
        )

        # 2. 결제 성공 확인
        if payment.is_paid():
            # 3. 주문 완료 처리
            complete_order(order_id, payment)
            return {"success": True, "payment": payment}
        else:
            return {"success": False, "error": "결제가 완료되지 않음"}

    except APIError as e:
        # 4. 에러 처리
        return {"success": False, "error": str(e)}
