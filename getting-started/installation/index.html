    설치

    TossPayments Python Server SDK를 설치하는 방법을 안내합니다.

    요구사항

    • Python: 3.9 이상
    • 의존성: requests>=2.28.0

    PyPI를 통한 설치

    가장 간단한 설치 방법입니다:

    pip install tosspayments-python-server-sdk

    가상환경 사용 권장

    프로젝트별로 독립적인 환경을 만들어 사용하는 것을 권장합니다:

    # 가상환경 생성
python -m venv venv

# 가상환경 활성화 (Linux/Mac)
source venv/bin/activate

# 가상환경 활성화 (Windows)
venv\Scripts\activate

# SDK 설치
pip install tosspayments-python-server-sdk

    Poetry를 사용한 설치

    Poetry를 사용하는 경우:

    poetry add tosspayments-python-server-sdk

    pipenv를 사용한 설치

    pipenv를 사용하는 경우:

    pipenv install tosspayments-python-server-sdk

    GitHub에서 직접 설치

    최신 개발 버전을 사용하고 싶다면:

    pip install git+https://github.com/jhwang0801/tosspayments-python-server-sdk.git

    특정 브랜치나 태그:

    # 특정 브랜치
pip install git+https://github.com/jhwang0801/tosspayments-python-server-sdk.git@develop

# 특정 태그
pip install git+https://github.com/jhwang0801/tosspayments-python-server-sdk.git@v1.0.1

    설치 확인

    설치가 완료되었는지 확인:

    import tosspayments_server_sdk
print(tosspayments_server_sdk.__version__)

    또는 간단한 테스트:

    from tosspayments_server_sdk import Client

# 클라이언트 생성 (테스트용 더미 키)
client = Client(secret_key="test_sk_dummy")
print("✅ SDK 설치 완료!")

    의존성 패키지

    SDK가 사용하는 주요 의존성 패키지들:

    • requests (>= 2.28.0): HTTP 클라이언트

    문제 해결

    SSL 인증서 문제

    일부 환경에서 SSL 인증서 문제가 발생할 수 있습니다:

    pip install --trusted-host pypi.org --trusted-host pypi.python.org --trusted-host files.pythonhosted.org tosspayments-python-server-sdk

    권한 문제 (Linux/Mac)

    권한 문제가 발생하는 경우:

    pip install --user tosspayments-python-server-sdk

    Python 버전 호환성

    Python 3.8 이하에서는 지원하지 않습니다. 버전 확인:

    python --version

    개발 환경 설정

    SDK 개발에 기여하고 싶다면:

    # 저장소 클론
git clone https://github.com/jhwang0801/tosspayments-python-server-sdk.git
cd tosspayments-python-server-sdk

# 개발용 설치
pip install -e .

# 개발 의존성 설치
pip install -r requirements-dev.txt

    버전 업데이트

    기존 설치된 SDK를 최신 버전으로 업데이트:

    pip install --upgrade tosspayments-python-server-sdk

    제거

    SDK를 제거하려면:

    pip uninstall tosspayments-python-server-sdk
