    빠른 시작

    이 가이드를 통해 5분 안에 토스페이먼츠 Python SDK를 시작할 수 있습니다.

    1. 설치

    pip install tosspayments-python-server-sdk

    2. API 키 준비

    토스페이먼츠 개발자센터에서 API 키를 발급받으세요.

    • 테스트용: test_sk_...로 시작
    • 운영용: live_sk_...로 시작

    주의사항

    API 키는 절대 클라이언트 사이드에 노출하지 마세요. 서버 환경변수로 관리하는 것을 권장합니다.

    3. 클라이언트 초기화

    from tosspayments_server_sdk import Client

# 환경변수에서 API 키 가져오기
import os
secret_key = os.getenv("TOSS_SECRET_KEY")

# 클라이언트 생성
client = Client(secret_key=secret_key)

# 테스트/운영 모드 확인
if client.is_test_mode:
    print("테스트 모드로 실행 중")

    4. 첫 번째 결제 승인

    결제는 일반적으로 다음 흐름으로 진행됩니다:

    1. 프론트엔드: 토스페이먼츠 결제창을 통해 결제 진행
    2. 결제창: 결제 완료 후 paymentKey, orderId, amount 반환
    3. 백엔드: 이 정보들로 결제 승인 API 호출
    try:
    # 결제 승인
    payment = client.payments.confirm(
        payment_key="mc_1234567890123456789012345678901234567890",
        order_id="order-20240101-001", 
        amount=15000
    )

    # 결제 성공
    if payment.is_paid():
        print(f"✅ 결제 완료!")
        print(f"주문번호: {payment.order_id}")
        print(f"결제금액: {payment.total_amount:,}원")
        print(f"결제방법: {payment.method}")

except Exception as e:
    print(f"❌ 결제 실패: {e}")

    5. 결제 정보 조회

    # paymentKey로 조회
payment = client.payments.retrieve("mc_1234567890123456789012345678901234567890")

# orderId로 조회  
payment = client.payments.retrieve_by_order_id("order-20240101-001")

print(f"결제 상태: {payment.status.value}")
print(f"잔여 금액: {payment.balance_amount:,}원")

    6. 결제 취소

    # 전체 취소
canceled_payment = client.payments.cancel(
    payment_key="mc_1234567890123456789012345678901234567890",
    cancel_reason="단순 변심"
)

# 부분 취소 (5,000원만)
canceled_payment = client.payments.cancel(
    payment_key="mc_1234567890123456789012345678901234567890", 
    cancel_reason="부분 환불",
    cancel_amount=5000
)

print(f"취소 완료: {canceled_payment.get_canceled_amount():,}원")

    7. 웹훅 처리

    웹훅을 통해 결제 상태 변경을 실시간으로 받을 수 있습니다:

    from flask import Flask, request
app = Flask(__name__)

@app.route('/webhook', methods=['POST'])
def handle_webhook():
    try:
        # 웹훅 파싱
        webhook_event = client.webhooks.verify_and_parse(request.data)

        if webhook_event.is_payment_event:
            payment = webhook_event.payment

            if payment.is_paid():
                # 결제 완료 처리
                print(f"결제 완료: {payment.order_id}")

            elif payment.is_canceled():
                # 결제 취소 처리  
                print(f"결제 취소: {payment.order_id}")

        return "OK", 200

    except Exception as e:
        print(f"웹훅 처리 실패: {e}")
        return "Error", 400
